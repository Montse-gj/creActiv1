# Nombre del Proyecto
crActiv
## Descripción
Plataforma donde encontrar y reservar talleres creativos (dibujo, cerámica, música, textil) son puntuales de gestión en unas horas o una jornada.
Está dirigido a tres roles de usuarios:
El usuario final que es el que compra el taller creativo
El profesor quien imparte el taller, puede o no disponer del espacio donde realizarlo
El centro espacio donde se realizará la actividad creativa.
Con esta plataforma se pretemde poner en contacto a centros y profesores para que pongan al alcance del usuario final la reserva y compra de la plaza para asistir al taller.
Con creActiv se busca solucionar la dificultad del usuario final de localizar un taller y la gestion de reserva, que mayoritariamente se tramita por RRSS, WhatsApp o web mal estructuradas y dificiles de encontrar.

## Estructura del Prototipo
Explicación de las vistas creadas y cómo se navega entre ellas.
* **index.html:** En la página principal el usuario final puede ver el listado de cursos, enlaces a login y registro y buscador de cursos
* **nombre_taller.html:** Esta página muestra la descripción del taller con precio, fechas disponibles, estado del curso y boton que lleva a la compra.
* **nombre_vista_3.html:** Descripción de esta tercera pantalla.
